Restaurant Table Booking Manager is a Java 8 desktop application developed using Swing and Maven following the MVC architectural pattern.

## Functionalities

- Add bookings
- Update bookings
- Delete bookings
- View bookings

Each booking contains:

- Booking ID
- Customer Name
- Table Number
- Booking Date
- Booking Time
- Special Request

## Application Features

- Booking management with complete CRUD operations
- MongoDB integration for data storage
- Test-Driven Development (TDD) approach
- Full code coverage analysis using SonarCloud and Coveralls
- Continuous Integration using GitHub Actions across multiple Java environments
